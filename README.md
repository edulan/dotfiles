# dotfiles

## Installation
Clone this repo in your HOME dir:
```bash
cd && git clone git@github.com:edulan/dotfiles.git .dotfiles
```

Then execute the installer:
```bash
cd .dotfiles && chmod u+x install.sh && ./install.sh
```
